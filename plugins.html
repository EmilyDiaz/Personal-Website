Figma Plugins

Dummy Data Generator

Dummy Data Generator allowed UX designers to quickly fill up table components in Figma with more accurate and unique data than just a typical lorem ipsum placeholder. Having unique placeholder text for different table columns helped us deliver more authentic prototypes to the developers, allowing us to illustrate different flows in which, for example, required different Veteran names or unique data for each veteran.

This plugin was created before Figma introduced its AI enhancments on the canvas, such as the replace-content feature. That feature rendered this plugin a little less useful, though the plugin still proved quicker to use than having to continuously correct the replace-content AI feature.

This was the first plugin I made. The figma documentation for its plugin API is a bit...sparse, so I did create my plugins with the help of AI, specifically Github Copilot's Claude model.

Prototype Variable Scanner

Variables in Figma are most often used for creating design tokens, but the UX team in the VA project I was in also utilized them for more advanced prototyping, which allowed us to implement conditionals on our prototypes and allowed for a reduction of how many frames are required to set up a useful prototyping.

One of the major cons to using variables over a vast number of frames to establish your prototypes is the lack of visibility. If Designer A were to come in and attempt to find what variables are used and where exactly they are being used on a file that Designer B has been working on, Designer A would have a tough time.

Prototype Scanner fixes that issue. The plugin scans the current page and outlines what variables are being used in prototypes on the page. It also tells users how many times a variable is used, what type of variable it is, and allows the user to jump to the exact frame (or frames, in the case of multiple use) the variable is used in. Enhancements for the future include optimization of results fetching, in order to increase the speed in which the page is scanned and results are returned.

If you have a figma account, find the plugin through the plugin menu on your figma canvas. Search for "Prototype variable scanner".

This plugin was not specific to the VA project, so it is available to download on Figma here.
